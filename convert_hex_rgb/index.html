How the Conversion Works

Step-by-Step Process:

  1. 1Remove the '#' symbol if present
  2. 2Convert 3-digit shorthand to 6-digit format
  3. 3Parse the hex string as a base-16 number
  4. 4Extract RGB values using bitwise operations

Bitwise Operations:

Red: (value >> 16) & 255
Green: (value >> 8) & 255
Blue: value & 255

Each color channel is extracted by shifting bits and masking with 255 (0xFF).
